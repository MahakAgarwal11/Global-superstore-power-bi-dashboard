# Global Superstore Power BI Dashboard

## Project Overview
Created an interactive Power BI dashboard using Global Superstore dataset to analyze sales, profit, customer segments, regional trends, and category performance.

## Tools Used
- Power BI
- Power Query
- DAX
- Excel

## Dashboard Features
- KPI Cards
- Sales Trend Analysis
- Category Performance
- Region Profit Analysis
- Country-wise Map
- Customer Segment Breakdown
- Interactive Slicers

## Key Insights
- Technology category generated highest sales.
- Consumer segment contributed highest revenue.
- Profit margin shows scope for improvement.
- Sales improved during later months.
